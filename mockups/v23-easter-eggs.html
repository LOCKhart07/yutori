Easter eggs (#79)

Two small, separate surprises. B passively appears on days the user actually spends nothing. D unlocks a hidden stats card after tapping the Build card's version row 7 times.

B — "No spends today" pill

Appears on the dashboard when today's actual spend is ₹0 and the user is viewing the current month. Info-tinted pill under the hero, with a small ✦ prefix so it reads as a quiet congratulation rather than a generic status.

B. Pill visible only on zero-spend days, current month. Disappears as soon as any transaction lands. Uses the same info-soft token as the override chips shipped in #144.

D — Tap Version 7× → lifetime stats card

On the About screen, tapping the Version row in the Build card 7 times reveals a hidden Yutori stats card directly below. Stays open for the rest of the session (or until the user leaves About). Taps on the Commit row don't count.

D3. After the 7th tap, a scrimmed dialog pops over the About screen with the stats grid. Dismiss via Close, back button, or tap outside. Values illustrative — ₹9.4L uses Indian short-form (lakh).

Pairing

The Zero-spend days cell in D3 is deliberately the same signal B marks — so the hidden card rewards observance of the quiet pill. The two features are technically independent but share a data path: a repo query for COUNT(DISTINCT date) of days in the current month where today's SMS produced no SPEND transactions (B), plus the same query lifetime-wide (D).
