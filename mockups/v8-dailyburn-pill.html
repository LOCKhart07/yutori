v8 · Daily-burn pill — pace tint + target (#16)

The third stat-pill slot in the dashboard adapts to the banner state: Normal → Daily burn, Over → Deficit, Approaching → Daily cap, OnTrack → Projected surplus, EarlyMonth/NoBudget → Elapsed days. #16 is scoped only to the Daily burn variant (i.e. DashboardBanner.Normal). Numbers below assume a ₹45k budget on a 30-day month so expected daily pace = ₹1,500/day. Pace tint reuses the hero's existing PaceBucket enum (Under / OnTrack / OverPace / Over).

Noted from review: when the banner is Over, the third pill shows "Deficit" — redundant with the "Budget exceeded · ₹X deficit carried over" banner right above it. Out of scope for #16 but worth a follow-up.
